# qiankun + Vite + React + Vue + Angular

qiankun 是一个基于 single-spa 的微前端实现库，旨在帮助大家能更简单、无痛的构建一个生产可用微前端架构系统;

此项目基于 qiankun + Vite（主应用）+ React（微应用） + Vue（微应用） + Angular（微应用）快速实现前端微服务
